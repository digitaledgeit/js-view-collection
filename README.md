# view-collection

A collection of views.

## Installation

CompomentJS:

    $ component install digitaledgeit/js-view-collection
    
    
NodeJS:
    
    $ npm install --save digitaledgeit-view-collection
    
    
## Building

ComponentJS:

    $ component build --dev

NodeJS:

    $ browserify -r assert:assert -r ./index.js:view-collection -r view:view > build/build.js
